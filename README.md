# emacs-ascii-cube

Rotating ASCII cube in Emacs buffer. <br>
Demo: [https://www.youtube.com/watch?v=KTEc1ZExRTU](https://www.youtube.com/watch?v=KTEc1ZExRTU)

# Install

Load ```ascii-cube.el``` or ```ascii-cube.elc``` (byte-compiled for speed).

```
M-x ascii-cube RET
```
